## 项目介绍

本项目为电子科技大学《Go语言与区块链技术》半期课设。

实现了一个简易的网页计算器，前端使用`HTML + CSS + Javascript`，后端使用`Golang`，前后端间通过`websocket`通信。

---

## 使用方法

- 一键启动：运行`run.bat`，自动打开服务器和网页。
- 分步启动：
  - 在`backend`目录下执行`go run main.go`；
  - 在浏览器中打开[本地端口](localhost:1234/frontend/)。
